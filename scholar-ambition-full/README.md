# 学术志：共同体 · 完整可玩原型

这是一个纯前端网页游戏原型，不需要安装依赖。主界面就是一张全屏 RPG 地图，HUD 和事件只作为悬浮层出现。

## 玩法

- 用 WASD 或上下左右方向键直接控制小人在“东海学术圈”地图上移动。
- 开局会出现引导界面，说明移动、互动、菜单和自动导航。
- 每月有 3 个行动点。
- 在办公室、图书馆、计算中心、学生工位、会议、期刊社、基金委、数据港口、研究前沿和咖啡厅之间跑图。
- 点击写着“进入”的入口节点，会自动走过去并切换到对应地图；站在入口旁按 `Enter` 也会直接进入。
- 子地图里点击“返回”节点，或站在返回点旁按 `Enter`，会直接回校园。
- 会议、期刊社、基金委、研究前沿会进入不同 RPG 地图，不再只是同一张校园图。
- 靠近 NPC 会触发学术战斗：组会质询、会议 Q&A、Reviewer 2、基金答辩、旧代码黑箱等。
- 新增更多人物：刘老师、数据管理员、Maria、校稿编辑、财务老师、复现者、旧范式守门人等都会触发不同战斗。
- 每次 NPC 战斗消耗 1 个行动点；胜利后的遭遇会标记为已解决，不能反复刷奖励。
- 战斗中可以用关键证据、逻辑重构、承认限制或强行扩大 claim 攻击，胜利会反映到论文、声望、可信度、经费和学生状态上。
- 按 `M` 可以查看剧情章节和任务列表；任务会随章节逐步解锁，要求你同时处理可信证据、学生压力、基金路线、合作边界、抢发风险和复现信誉。
- 游戏现在有主线过场、地图初见剧情、关键战斗后的角色对白，以及会改变数值的剧情选择。
- 校园主地图已替换为无内嵌人物版本；NPC 和怪物使用独立透明 sprite，不再把人物画死在背景里。
- 推进论文的五条线：问题、方法、证据、写作、可信。
- 经营经费、声望、信誉、士气、领域影响力和学生状态。
- 触发投稿、大修、rebuttal、基金申请、合作邮件、匿名质疑、竞争组抢发、数据协议、学生低谷、学院考核、资格考试、复现追问和最终评审。
- 难度已上调：开局经费和信誉更低，投稿/基金阈值更高，月度成本、学生压力、合作债和战斗反击都更重。
- 目标是在 36 个月内发表 3 篇可信论文，并尽量建立一个健康的研究共同体。

## 自动导航

按 `M` 打开研究菜单后，可以看到“自动导航”面板并选择策略：

- 均衡
- 冲论文
- 保经费
- 低风险
- 带学生

点击“自动导航一步”会让小人自动移动到合适地点并执行下一步行动。

点击“自动推进”会连续执行，直到出现事件弹窗、游戏结束或玩家手动停止。

## 键盘

- `WASD` / `方向键`：直接移动小人。
- `Enter`：靠近地点后打开 RPG 对话框；靠近 NPC 时优先触发对话或战斗。
- `Space`：自动导航一步。
- `M`：打开/关闭研究菜单。
- `Esc`：关闭对话框或菜单。

## 运行

直接打开：

```text
scholar-ambition-full/index.html
```

或者在项目根目录启动静态服务：

```bash
python3 -m http.server 4174
```

然后访问：

```text
http://127.0.0.1:4174/scholar-ambition-full/index.html
```
