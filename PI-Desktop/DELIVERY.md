# vibe科研模拟器 · 交付说明

## 版本定位

这是一个可直接打开运行的静态网页游戏原型。

- 形态：单页前端
- 运行：直接打开 `index.html`
- 存档：浏览器 `localStorage`
- 目标：做成一个短局、高重玩度、黑色幽默的科研管理模拟器

## 当前可玩内容

- 假 Claude / Codex 风格科研操作台
- 3 actions per week 的短回合结构
- 模型市场的隐藏情报系统
- tokens / compute 分离
- 多学科项目池
- 招人、安抚、内审、切片灌水、投稿、基金
- 成员 trait 事故系统
- 随机事件扩容
- 多结局
- 战报复制与本地存档
