# vibe科研操作系统

一个纯前端网页游戏，主题是“假的科研 agent 操作系统 / vibe科研模拟器”。

这一版刻意做成广义科研场景，不绑定任何单一学科。当前内容覆盖：

- 现金买 tokens
- 模型市场：供应商宣传、组内口碑和实测踩坑一起决定你对模型的认识
- 新手引导层：推荐下一步、四步玩法说明、按用途分组的动作区
- tokens 和 compute 分离：想法和实际计算分别烧不同资源
- 启动模式：Career / 每日挑战 / 速冲局
- 开局流派：爆肝 PI / 算力投机客 / 温柔导师
- 文献扫描、代码原型、写论文、人工核验、跑 benchmark、申请基金、内部审计、切片灌水、休息、投稿
- 团队压力和安抚动作
- 招人进组、实验室成员卡、坑人 trait、周结算事故和造假风险
- 随机事件扩容：挂名作者、引用绑架、掠夺性会议、论文中介、开放数据审查、AI 审稿骚操作
- 多学科项目切换：气候、生命科学、数字人文、AI、材料、社科
- 核心资源：现金、tokens、compute、judgment、reputation、mental HP、hallucination risk、team stress、contract clock
- 假终端日志和系统事件流
- 本地存档、继续 run、多个结局、可复制战报

## 运行

直接打开 `index.html` 即可。存档使用浏览器 `localStorage`。
